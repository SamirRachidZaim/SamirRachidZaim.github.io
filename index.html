BinomialRF: Developing Machine Learning Modules to detect feature interactions

I developed a novel machine learning algorithms to answer the basic question: if I know some features are important, which combinations of these are important together, and can I quantify how important these combinations are? I led this research from start to finish, including the statistical derivation, simulation studies, manuscript writing, and developed the accompanying machine learning software package in both R & python.
